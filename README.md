# Embodied Intelligence Playbook

A curated open-source playbook for embodied intelligence: roadmaps, papers, practical checklists, and resources.

## What is this?

This repository is a structured, open-source knowledge base for embodied intelligence. It focuses on:

- research roadmaps across major subfields
- paper collections organized by topic and top venues
- practical checklists for learning, reproduction, and project building
- resources including datasets, simulators, frameworks, and benchmarks

## Who is this for?

- students entering embodied AI
- researchers building a direction map
- engineers doing reproduction and project implementation
- reading groups and lab-internal knowledge sharing

## Main Topics

- Vision-Language-Action
- World Models
- Manipulation
- Grasping
- Affordance Learning
- 3D Perception for Robotics
- Navigation
- Sim2Real
- Benchmarks and Evaluation
