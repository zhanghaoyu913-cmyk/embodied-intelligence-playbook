# Frameworks

A good framework can reduce friction. A bad framework can silently shape the entire project in the wrong way.

## What will be collected here?

- robot learning codebases
- simulation toolkits
- manipulation-oriented stacks
- evaluation harnesses
- VLA or foundation-model-oriented embodied frameworks
- practical utilities for experiment logging, data handling, and benchmarking

## Selection criteria

Frameworks featured in this repository should be useful under at least one of the following conditions:

- strong reproduction value
- clear educational value
- stable engineering foundation
- relevance to current embodied AI directions
- good bridge between research and system building
