# Datasets

This page will collect datasets that are especially useful for **embodied AI practice** rather than only for citation completeness.

## How this page will be organized

### By task family
- manipulation
- grasping
- navigation
- language-conditioned robotics
- affordance understanding
- 3D object and scene assets

### By practical concern
- scale
- sensor modalities
- license and accessibility
- quality of annotations
- ease of starting a first project
- suitability for simulation or real-world transfer

## What makes a dataset useful here?

The most useful datasets are not always the largest.  
For this repository, a strong dataset is one that helps answer at least one of these questions:

- Can I reproduce a known system with it?
- Can it support a focused project with clear evaluation?
- Does it expose a non-trivial embodied challenge such as contact, language grounding, or long-horizon task structure?
- Is it realistic enough to matter, but accessible enough to begin with?
