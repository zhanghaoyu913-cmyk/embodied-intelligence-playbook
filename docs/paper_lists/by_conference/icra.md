# ICRA

## Why watch ICRA

ICRA is the broadest mainstream robotics venue. It is where manipulation, teleoperation, grasping, system integration, and benchmark-building often appear in the same year.

## Official entry

https://www.ieee-ras.org/conferences-workshops/fully-sponsored/icra

## Curation note

For ICRA 2025, public discovery is uneven: some strong papers have clean project pages, while others are easiest to access via official workshop-sidecar OpenReview pages. This list mixes both, and labels the workshop-sidecar entries clearly.

## 2025 embodied AI paper picks

- [DexMimicGen: Automated Data Generation for Bimanual Dexterous Manipulation via Imitation Learning](https://dexmimicgen.github.io/)
  ICRA 2025. Strong pick if you care about scaling bimanual dexterous data from a small number of seed demonstrations.

- [ForceMimic: Force-Centric Imitation Learning with Force-Motion Capture System for Contact-Rich Manipulation](https://forcemimic.github.io/)
  ICRA 2025. Worth reading because it treats force as first-class training signal rather than hidden execution detail.

- [Point Policy: Unifying Observations and Actions with Key Points for Robot Manipulation](https://openreview.net/forum?id=Rk2gDxVUfq)
  ICRA 2025 workshop-sidecar. Useful for learning from human videos with a cleaner intermediate representation than raw image-to-action pipelines.

- [Human2LocoMan: Learning Versatile Quadrupedal Manipulation with Human Pretraining](https://openreview.net/forum?id=6RwcBFg3m4)
  ICRA 2025 workshop-sidecar. A good read if you want cross-embodiment pretraining and loco-manipulation instead of fixed-base manipulation only.

- [BEHAVIOR Robot Suite: Streamlining Real-World Whole-Body Manipulation for Everyday Household Activities](https://openreview.net/forum?id=e1zNSeIOGy)
  ICRA 2025 workshop-sidecar. Useful because it is not just another policy paper; it is a whole-body household manipulation stack.
