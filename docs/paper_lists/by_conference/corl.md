# CoRL

## Why watch CoRL

CoRL is still the most directly robot-learning-centric venue. If you want papers that are close to data, policy design, teleoperation, world models, and deployable manipulation stacks, CoRL is usually the first place to look.

## Official entry

https://www.corl.org/

## Curation note

This page emphasizes 2025 papers that complement the roadmap pages instead of repeating the most obvious headline systems.

## 2025 embodied AI paper picks

- [FLOWER: Democratizing Generalist Robot Policies with Efficient Vision-Language-Flow Models](https://openreview.net/forum?id=JeppaebLRD)
  CoRL 2025. A good paper to track if you care about smaller and cheaper VLA-style policies instead of only giant model scaling.

- [Long-VLA: Unleashing Long-Horizon Capability of Vision Language Action Model for Robot Manipulation](https://openreview.net/forum?id=irh5o90Mj1)
  CoRL 2025. Useful because it explicitly targets long-horizon failure modes and introduces L-CALVIN rather than staying in short-horizon tabletop settings.

- [ParticleFormer: A 3D Point Cloud World Model for Multi-Object, Multi-Material Robotic Manipulation](https://proceedings.mlr.press/v305/huang25c.html)
  CoRL 2025. A strong pick for readers tracking 3D world models that actually interact with rigid, deformable, and mixed-material scenes.

- [AgentWorld: An Interactive Simulation Platform for Scene Construction and Mobile Robotic Manipulation](https://proceedings.mlr.press/v305/zhang25f.html)
  CoRL 2025. More useful than a single method paper if you want a simulation and data ecosystem for household mobile manipulation.

- [DexUMI: Using Human Hand as the Universal Manipulation Interface for Dexterous Manipulation](https://proceedings.mlr.press/v305/xu25b.html)
  CoRL 2025. Worth tracking because it attacks the dexterous-data bottleneck through human-hand-as-interface rather than only better policies.

- [CLONE: Closed-Loop Whole-Body Humanoid Teleoperation for Long-Horizon Tasks](https://proceedings.mlr.press/v305/li25h.html)
  CoRL 2025. High-signal paper if you are watching humanoid data collection and whole-body teleoperation rather than only arm-centric manipulation.
