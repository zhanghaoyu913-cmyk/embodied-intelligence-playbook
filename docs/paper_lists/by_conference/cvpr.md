# CVPR

## Why watch CVPR

CVPR is one of the best places to track perception-side progress that later becomes embodied capability: affordance grounding, video-to-action transfer, scene generation, and robot-facing visual pretraining.

## Official entry

https://cvpr.thecvf.com/

## 2025 embodied AI paper picks

- [VidBot: Learning Generalizable 3D Actions from In-the-Wild 2D Human Videos for Zero-Shot Robotic Manipulation](https://openaccess.thecvf.com/content/CVPR2025/html/Chen_VidBot_Learning_Generalizable_3D_Actions_from_In-the-Wild_2D_Human_Videos_CVPR_2025_paper.html)
  CVPR 2025. Good paper for the "learn from human video, not robot data only" line.

- [DynScene: Scalable Generation of Dynamic Robotic Manipulation Scenes for Embodied AI](https://openaccess.thecvf.com/content/CVPR2025/html/Lee_DynScene_Scalable_Generation_of_Dynamic_Robotic_Manipulation_Scenes_for_Embodied_CVPR_2025_paper.html)
  CVPR 2025. Worth reading if you care about synthetic scene and trajectory generation as a data engine for robot learning.

- [Robotic Visual Instruction](https://openaccess.thecvf.com/content/CVPR2025/html/Li_Robotic_Visual_Instruction_CVPR_2025_paper.html)
  CVPR 2025. Interesting because it questions whether text alone is the right interface for robot instruction and proposes a more spatially precise visual interface.

- [Mitigating the Human-Robot Domain Discrepancy in Visual Pre-training for Robotic Manipulation](https://openaccess.thecvf.com/content/CVPR2025/html/Zhou_Mitigating_the_Human-Robot_Domain_Discrepancy_in_Visual_Pre-training_for_Robotic_CVPR_2025_paper.html)
  CVPR 2025. A useful complement to larger policy papers because it focuses on the representation-transfer problem directly.

- [SeqAfford: Sequential 3D Affordance Reasoning via Multimodal Large Language Model](https://openaccess.thecvf.com/content/CVPR2025/html/Yu_SeqAfford_Sequential_3D_Affordance_Reasoning_via_Multimodal_Large_Language_Model_CVPR_2025_paper.html)
  CVPR 2025. Strong pick if you want affordance work that explicitly moves from single-step to sequential reasoning.
