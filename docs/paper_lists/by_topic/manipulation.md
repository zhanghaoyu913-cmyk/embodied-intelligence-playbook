# Manipulation Paper & Project List

## 2025–2026 frontier

- **[Long-VLA](https://long-vla.github.io/)** — CoRL 2025; long-horizon VLA for robot manipulation.
- **[AgentWorld](https://yizhengzhang1.github.io/agent_world/)** — CoRL 2025; interactive simulation platform for household mobile manipulation.
- **[SAM2Act](https://sam2act.github.io/)** — ICML 2025; visual foundation model plus memory architecture for manipulation.
- **[DexUMI](https://proceedings.mlr.press/v305/xu25b.html)** — CoRL 2025; human hand as a universal manipulation interface for dexterous manipulation.
- **[Dex1B](https://www.roboticsproceedings.org/rss21/p106.html)** — RSS 2025; billion-scale generated demonstrations for dexterous manipulation.
- **[Enhancing Policy Learning with World-Action Model](https://arxiv.org/abs/2603.28955)** — 2026 preprint; evaluates action-regularized WAM representations on CALVIN manipulation tasks.
