# Affordance Paper & Project List

## 2025–2026 frontier

- **[SeqAfford](https://seq-afford.github.io/)** — CVPR 2025; sequential 3D affordance reasoning via multimodal large language models.
- **[Robotic Visual Instruction](https://openaccess.thecvf.com/content/CVPR2025/html/Li_Robotic_Visual_Instruction_CVPR_2025_paper.html)** — CVPR 2025; visual instruction interface for spatially precise robotic guidance.
- **[BiAssemble](https://proceedings.mlr.press/v267/shen25i.html)** — ICML 2025; collaborative bimanual affordance for geometric assembly.
- **[RoboInter](https://github.com/InternRobotics/RoboInter)** — ICLR 2026; intermediate representations including affordance boxes, grasp poses, contact points, and motion traces.
