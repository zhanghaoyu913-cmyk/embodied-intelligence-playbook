# Grasping Paper List

Curated starting list for robotic grasping.

| Paper | Venue / Year | Focus | Links |
|---|---|---|---|
| GPD: Grasp Pose Detection in Point Clouds | 2017 | classic point-cloud grasp detection | [Paper](https://arxiv.org/abs/1706.09911) · [Code](https://github.com/atenpas/gpd) |
| GraspNet-1Billion | CVPR 2020 | large-scale general object grasping benchmark | [Project](https://graspnet.net/) |
| Contact-GraspNet | ICRA 2021 | direct 6-DoF grasp generation in cluttered scenes | [Paper](https://research.nvidia.com/publication/2021-03_contact-graspnet-efficient-6-dof-grasp-generation-cluttered-scenes) · [Code](https://github.com/NVlabs/contact_graspnet) |
| Graspness Discovery in Clutters | ICCV 2021 | dense graspability reasoning in clutter | [Publications page](https://graspnet.net/publications.html) |
| AnyGrasp | T-RO 2023 | practical robust dense grasp library | [Project](https://graspnet.net/anygrasp.html) |
| SuctionNet-1Billion | RA-L 2021 | suction grasp benchmark | [Publications page](https://graspnet.net/publications.html) |
