# Academic Labs & Universities

This page is not a ranking. It is a **navigation map** for readers who want to follow influential academic groups, open projects, and institutional entry points.

| Institution | Why follow | Representative entries |
|---|---|---|

## How to use this page

- Follow **Stanford / Berkeley** if you want reproducible manipulation and benchmark ecosystems.
- Follow **CMU** if you want broad embodied-AI systems and institutional depth.
- Follow **Tsinghua / PKU / SJTU / ZJU / Fudan** if you want the fast-moving China-side embodied and humanoid ecosystem.
- Follow the project pages first, then the papers, then the institution.

## Notes

Some institutions expose polished public project pages, while others expose mostly center pages, challenge pages, or news pages. This page prioritizes the most official public entry point available.
