## How to use this page

- Follow **Google DeepMind / NVIDIA / ByteDance / Alibaba** for model releases and toolchains.
- Follow **Meta** when you care about world-model style foundations.
- Follow **Meituan** for deployment-oriented logistics robotics.
- Follow **OpenAI** for historical RL and sim-to-real context that still shapes many modern ideas.
