# Conferences & Journals

This page expands the venue coverage of the repository so readers know where embodied-AI work tends to appear.

## Core conferences to watch

| Venue | Why it matters | Link |
|---|---|---|
| ICRA | broadest mainstream robotics venue | https://www.ieee-ras.org/conferences-workshops/fully-sponsored/icra |
| IROS | large robotics systems and applications venue | https://www.ieee-ras.org/conferences-workshops/financially-co-sponsored/iros |
| RSS | highly selective robotics research venue | https://roboticsconference.org/ |
| CoRL | machine learning for robotics | https://www.corl.org/ |
| CVPR | perception-heavy embodied and robot-vision work | https://cvpr.thecvf.com/ |
| ICCV | perception, 3D vision, and multimodal work relevant to embodied AI | https://iccv.thecvf.com/ |
| ECCV | 3D scene understanding, video, perception, affordance, VLMs | https://eccv.ecva.net/ |
| ICLR | foundation models, RL, world models, embodied reasoning | https://iclr.cc/ |
| ICML | learning systems, world models, RL, large-scale training | https://icml.cc/ |
| NeurIPS | RL, multimodal systems, large-scale models, benchmarks | https://neurips.cc/ |

## Core journals to watch

| Journal | Why it matters | Link |
|---|---|---|
| IEEE RA-L | fast-turnaround robotics journal used by much of the community | https://www.ieee-ras.org/publications/ra-l |
| IEEE T-RO | major archival robotics journal | https://www.ieee-ras.org/publications/t-ro |
| IJRR | influential long-form robotics journal | https://journals.sagepub.com/home/ijr |
| Science Robotics | high-impact robotics showcase venue | https://www.science.org/journal/scirobotics |
| Nature / Nature Machine Intelligence | occasional but important embodied-AI milestones | https://www.nature.com/ |

## Practical advice

- For **VLA and multimodal embodied systems**, watch ICLR, ICML, NeurIPS, CVPR, ICCV, and CoRL.
- For **grasping, manipulation, and robot systems**, watch ICRA, RSS, IROS, and RA-L.
- For **long-form robotics results**, watch T-RO and IJRR.
